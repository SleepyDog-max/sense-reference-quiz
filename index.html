Sense and Reference Quiz

1) The sense of a referring expression can change depending on the context in which it is used.

2) A referring expression always picks out the same referent, regardless of who is speaking or the situation.

3) Which of the following statements is TRUE about the difference between sense and reference?

4) Which of the following is an example of a referring expression with a variable reference?

Quiz Summary

Explanation:

1) False - The **sense** of an expression is its stable, dictionary meaning.

2) False - A referring expression’s reference can change based on context.

3) Correct answer: (a) - **Sense** is the literal meaning, **reference** depends on context.

4) Correct answer: (c) - *"My teacher"* refers to different people based on context.
